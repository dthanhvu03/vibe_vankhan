## 1. Chiến lược tổng thể

**Traffic model chính**: Organic search (Google, ~95%).
**Vì sao**: user chủ động search khi cần cúng lễ — intent cực rõ, conversion cao.
**Trade-off**: SEO cần 3-6 tháng để rank → 3 tháng đầu traffic thấp là bình thường.

**Chiến lược 3 lớp**:
1. **Head keyword** (`văn khấn mùng 1`, `văn khấn nhập trạch`) — cạnh tranh cao, dài hạn
2. **Body keyword** (`văn khấn mùng 1 hàng tháng`, `cúng nhập trạch nhà chung cư`) — cạnh tranh vừa, target tháng 2-6
3. **Long-tail** (`văn khấn mùng 1 tháng 8 âm 2026 chuẩn nhất`) — cạnh tranh thấp, có thể rank sớm

Ưu tiên phase 1: **body + long-tail** để có traffic sớm.

### 2.2 Body keyword (2-4 từ, dài hơn head)

Mỗi bài văn khấn nên target **1 head + 3-5 body**:

**Ví dụ cho bài "Văn khấn mùng 1 hàng tháng"**:
- Head: `văn khấn mùng 1`
- Body: `văn khấn mùng 1 hàng tháng`, `văn khấn thần linh mùng 1`, `văn khấn gia tiên mùng 1`, `cúng mùng 1 hàng tháng cần chuẩn bị gì`

**Ví dụ cho bài "Văn khấn nhập trạch"**:
- Head: `văn khấn nhập trạch`
- Body: `văn khấn cúng nhà mới`, `văn khấn về nhà mới`, `văn khấn nhập trạch nhà chung cư`, `cúng nhập trạch cần những gì`

### 2.3 Long-tail (5+ từ, intent siêu cụ thể)

Đây là **cơ hội lớn nhất phase 1** — big site không cover hết:

**Long-tail templates**:
- `văn khấn [dịp] chuẩn nhất [năm]` (evergreen theo năm)
- `văn khấn [dịp] miền [Bắc/Trung/Nam]`
- `văn khấn [dịp] tại nhà`
- `cúng [dịp] cần chuẩn bị những gì`
- `[dịp] nên cúng vào giờ nào`
- `mâm cúng [dịp] gồm những gì`
- `cách khấn [dịp] đơn giản`

**Ví dụ cụ thể**:
- "văn khấn cô hồn mùng 2 hàng tháng chuẩn nhất"
- "văn khấn nhập trạch chung cư ngắn gọn"
- "cúng ông Táo 2026 vào giờ nào tốt" (chỉ ranh giới, không sa vào bói)
- "văn khấn Vu Lan cho người trẻ mới lập gia đình"

### 2.4 Question keyword (cho FAQ + blog)

- "cúng mùng 1 vào giờ nào tốt?"
- "có cúng rằm chay được không?"
- "cúng đầy tháng bé gái khác bé trai như nào?"
- "quên cúng ông táo có sao không?"

**Chiến thuật**: Trả lời trong section FAQ của bài văn khấn liên quan, đánh dấu schema `FAQPage` → có cơ hội xuất hiện trong Google FAQ rich result.

## 4. On-page SEO checklist

Cho mỗi trang văn khấn:

- [ ] URL sạch: `/van-khan/{slug}` (không dấu, có keyword)
- [ ] `<title>` 50-60 ký tự chứa keyword chính
- [ ] `<meta description>` 140-160 ký tự có CTA
- [ ] `<h1>` 1 duy nhất chứa keyword
- [ ] `<h2>` `<h3>` phân cấp đúng
- [ ] Keyword density 1-2% (không nhồi)
- [ ] Alt text đầy đủ cho mọi ảnh
- [ ] Internal link 3-5 văn khấn/blog liên quan
- [ ] External link (nếu có): `rel="noopener"`, không quá 2
- [ ] Schema markup:
  - [ ] `Article`
  - [ ] `HowTo` (cho phần chuẩn bị)
  - [ ] `FAQPage` (cho phần FAQ)
  - [ ] `BreadcrumbList`
- [ ] Open Graph + Twitter Card
- [ ] Canonical URL
- [ ] Word count > 800
- [ ] Lighthouse SEO score > 95
- [ ] Mobile-friendly (test Google Mobile-Friendly Test)
- [ ] Page speed LCP < 2.5s (Vercel Analytics)

## 5. Technical SEO (kết hợp với SRS §4.2)

### 5.1 Sitemap

Auto-generate từ MDX files, submit qua Google Search Console + Bing Webmaster.

```
/sitemap.xml
├── / (trang chủ)
├── /van-khan/... (tất cả 15+)
├── /goc-phong-tuc/... (tất cả blog)
├── /lich-am
└── /le-tet, /hang-thang, /gia-dinh, /kinh-doanh (category)
```

Priority setting:
- Trang chủ: 1.0
- Văn khấn: 0.8
- Category: 0.7
- Blog: 0.6
- About/Privacy: 0.3

`changefreq`:
- Trang chủ: weekly
- Văn khấn: monthly
- Blog: yearly

### 5.2 Robots.txt

```
User-agent: *
Allow: /
Disallow: /api/
Disallow: /admin/

Sitemap: https://vankhanviet.com/sitemap.xml
```

### 5.3 Schema JSON-LD template

Trên page văn khấn:

```json
{
  "@context": "https://schema.org",
  "@graph": [
    {
      "@type": "Article",
      "headline": "Văn khấn mùng 1 hàng tháng",
      "datePublished": "2026-07-07",
      "dateModified": "2026-07-15",
      "author": { "@type": "Organization", "name": "Văn Khấn Việt" },
      "image": "https://vankhanviet.com/images/mam-cung/mung-1.jpg"
    },
    {
      "@type": "HowTo",
      "name": "Cách cúng mùng 1 hàng tháng",
      "step": [
        {"@type": "HowToStep", "name": "Chuẩn bị mâm cúng", "text": "..."},
        {"@type": "HowToStep", "name": "Đọc văn khấn", "text": "..."}
      ]
    },
    {
      "@type": "FAQPage",
      "mainEntity": [
        {
          "@type": "Question",
          "name": "Cúng mùng 1 vào giờ nào tốt?",
          "acceptedAnswer": { "@type": "Answer", "text": "..." }
        }
      ]
    }
  ]
}
```

### 5.4 URL redirects

- `vankhanviet.com` (không có `www`) là canonical
- Redirect `www.vankhanviet.com` → `vankhanviet.com` (301)
- Redirect `http` → `https` (Vercel default)
- Redirect trailing slash cho consistency (chọn 1 pattern)

### 5.5 Page speed budget

| Metric | Target | Tool đo |

| TTFB | < 800ms | WebPageTest |

## 6. Backlink strategy

### 6.1 Phase 1 (Tháng 1-3): Passive & organic

**Không mua link.** Chỉ:

- **Chia sẻ tự nhiên** trong 5-10 group FB target:
  - "Cộng đồng mẹ bỉm sữa" (đầy tháng, thôi nôi)
  - "Người mua nhà đầu tiên" (nhập trạch)
  - "Chủ shop online" (khai trương, thần tài)
  - "Hà Nội - Việc làm & Cuộc sống" (cúng cô hồn, Vu Lan)
- **Comment blog liên quan** khi có giá trị (không spam), để link natural khi phù hợp
- **Trả lời câu hỏi Voz, Reddit VN, r/VietNam**: hỏi văn khấn → trả lời có link
- **Đăng đàn Otofun, Webtretho, Lamchame** — các cộng đồng lớn có age domain trust cao

### 6.2 Phase 2 (Tháng 4-6): Active outreach

- **Guest post** 2-3 bài trên blog liên quan (VnExpress góc phong tục, Kênh 14 sức khoẻ đời sống nếu có cơ hội)
- **Partnership** 1-2 shop đồ thờ (đổi link tự nhiên)
- **PR nhẹ**: pitch story cho báo mạng — "Ứng dụng số hoá văn hoá truyền thống của 1 dev trẻ"
- **Interview thầy chùa/nhà nghiên cứu văn hoá** — họ share có credibility

### 6.3 KPI backlink

| Tháng | Target unique referring domains |
|---|---|
| Tháng 1 | 5-10 |
| Tháng 3 | 20-30 |
| Tháng 6 | 50-80 |
| Tháng 12 | 100-200 |

**Chất > lượng**. 1 backlink từ vnexpress.net > 100 backlink forum spam.

2. **2-3 tuần trước**: publish bài "10 câu hỏi thường gặp về [sự kiện]"
